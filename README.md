# GDG Quiz Website

A simple quiz website built with React and React Router.

## Features

- Google green (#34A853) color scheme
- Mobile responsive design
- Invalid path handling
- Ready for Vercel deployment

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment to Vercel

1. Push this repository to GitHub
2. Import the project in Vercel
3. Deploy (no additional configuration needed - `vercel.json` handles SPA routing)

## Tech Stack

- React 18
- React Router 6
- Vite
- CSS3
