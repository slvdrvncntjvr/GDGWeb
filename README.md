# GDG Quiz Website

A simple quiz website built with React and React Router, styled with Google colors.

## Features

- 5 question pages with unique URL paths
- Google green (#34A853) color scheme
- Mobile responsive design
- Invalid path handling
- Ready for Vercel deployment

## Question Paths

- `/x9k2mfp7qw` - Question 1
- `/n4bt8zj5vh` - Question 2
- `/k3pm9wq5rt` - Question 3
- `/b6nz4xc8vy` - Question 4
- `/m2qf7ht9jw` - Question 5

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment to Vercel

1. Push this repository to GitHub
2. Import the project in Vercel
3. Deploy (no additional configuration needed - `vercel.json` handles SPA routing)

## Tech Stack

- React 18
- React Router 6
- Vite
- CSS3
